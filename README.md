# 📚 Study Planner Web App

A simple and clean **Study Planner Web App** to help you organize your daily tasks and stay consistent.

---

## ✨ Features

* ➕ Add tasks with date
* 📅 Schedule your study plan
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 🧹 Clear all tasks
* 🎨 Beautiful pink-themed UI
